---
title: Demo App
---

The demo application lives in `examples/demo` and is deployed at:

- https://ambientcss.vercel.app/

Run it from the monorepo root:

```bash
pnpm --filter demo dev
```

Build it:

```bash
pnpm --filter demo build
```

Docs are deployed separately to GitHub Pages:

- https://kikkupico.github.io/ambientcss/
