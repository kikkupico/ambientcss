Concept

Physical lighting, orthographic view

Ambient CSS imagines UI as a front-facing object lit by a key and fill source. Elevation, thickness, and edge profiles sculpt the shadows. Emitters glow instead of casting a shadow. Finishes shape how surfaces react to light.

Key variables
--amb-light-x / --amb-light-y
--amb-key-light-intensity
--amb-fill-light-intensity
Orthographic view

Elements are shaded as if viewed straight-on, giving predictable shadow direction and consistent depth.

Two light rig

A key light drives shadow length while a softer fill lifts the darkest areas.

Depth + edges

Stack elevation and thickness to create layered panels, fillets, chamfers, and grooves.

Emitter + finish

Emitters glow instead of casting shadows, while finishes tune surface sheen.
