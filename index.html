Variable Reference

AMB-VAR

Customize the lighting environment using CSS custom properties. All values can be animated for dynamic lighting transitions.

Variable Default Description
--amb-light-x 1 Light direction X (-1 to 1)
--amb-light-y -1 Light direction Y (-1 to 1)
--amb-key-light-intensity 0.3 Main light strength (0 to 1)
--amb-fill-light-intensity 0 Fill light strength (0 to 1)
--amb-elevation 0 Element elevation (0 to 3)
--amb-thickness 0 Element thickness (0 to 3)
--amb-emit-color 255, 200, 100 RGB values for emitter glow
--amb-emit-intensity 0.5 Emitter brightness (0 to 1)

Quick Start

AMB-QST

Get started with Ambient CSS in seconds. Include the stylesheet and start adding classes to your elements.

1. Include the CSS
<link href="index.css" rel="stylesheet">
2. Add Ambient Classes
<div class="ambient amb-fillet amb-elevation-2">
  Your content here
</div>
3. Customize Lighting (Optional)
:root {
  --amb-light-x: -1; /* Light from left */
  --amb-key-light-intensity: 0.5; /* Brighter */
}
