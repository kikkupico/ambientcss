Documentation extra

Use the .ambient class on any element, then layer in modifiers:

  • Edges: .amb-fillet, .amb-chamfer, .amb-groove
  • Depth: .amb-elevation-1-4, .amb-thickness-1-3
  • Finishes: .amb-matte, .amb-shiny, .amb-translucent
  • Emitters: .amb-emitter for glowing UI accents

Override any lighting variable directly in CSS for custom rigs. Pair with the included buttons, toggles, and readouts for a full synth interface toolkit.
