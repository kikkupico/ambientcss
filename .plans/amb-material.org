#+TITLE: Ambient Material Options (amb-mat)
#+AUTHOR: Gemini CLI
#+DATE: 2026-03-31

* Goal
Implement a new set of utility classes =amb-mat-*= to define the material properties of an element, affecting its background and texture while remaining consistent with the global ambient lighting model.

* Requirements
- Default: =amb-mat-matte=
- Options:
  - =amb-mat-matte=: Standard solid/flat skeuomorphic surface.
  - =amb-mat-shiny=: Adds a light-reflective gradient.
  - =amb-mat-glass=: Adds translucency and a background blur effect (frosted glass).

* Proposed Implementation

** CSS Definitions (packages/ambient-css/src/ambient.css)

*** Matte (Default)
The default material will use the current surface logic.
#+BEGIN_SRC css
.amb-mat-matte {
  /* Default behavior, standard background-color */
}
#+END_SRC

*** Shiny
Utilizes a gradient that aligns with the light direction (=--amb-light-x=, =--amb-light-y=).
#+BEGIN_SRC css
.amb-mat-shiny {
  --_shiny-angle: calc(atan2(var(--amb-light-y), var(--amb-light-x)) + 270deg);
  background-image: linear-gradient(
    var(--_shiny-angle),
    hsl(var(--amb-light-hue) var(--amb-light-saturation) 100% / 0.5) 0%,
    hsl(var(--amb-light-hue) var(--amb-light-saturation) 100% / 0.05) 50%,
    hsl(var(--amb-light-hue) var(--amb-light-saturation) 0% / 0.1) 100%
  );
  background-blend-mode: soft-light;
}
#+END_SRC

*** Glass
Frosted glass effect using =backdrop-filter=.
#+BEGIN_SRC css
.amb-mat-glass {
  background-color: hsl(var(--amb-light-hue) var(--amb-light-saturation) 100% / 0.1);
  backdrop-filter: blur(12px) saturate(160%);
  -webkit-backdrop-filter: blur(12px) saturate(160%);
  border: 1px solid hsl(var(--amb-light-hue) var(--amb-light-saturation) 100% / 0.2);
}
#+END_SRC
